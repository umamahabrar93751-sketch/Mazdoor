# Mazdoor
Website on hiring labours/common workers 

## 📋 About
*Mazdoor.com* is a bilingual platform that connects daily wage workers (Mazdoors) with employers in Pakistan. The platform facilitates instant hiring for construction helpers, electricians, plumbers, cleaners, and general laborers.
---

## ✨ Features

### 👷 For Workers
- Real-time job alerts in your area
- Accept/reject jobs with one click
- Direct chat with employers
- Payment via EasyPaisa, JazzCash, or Card
- Track job history and earnings

### 🏢 For Employers
- Post jobs instantly with details
- Manage active job listings
- View worker profiles and ratings
- Direct communication with workers
- Access previous worker history

### 👑 For Admins
- Platform overview with statistics
- Verify/suspend users
- Moderate job postings
- Handle disputes and reports
- View analytics and trends
- Configure platform settings

### 🌐 Platform Features
- *Bilingual* (English/Urdu) toggle
- *Dark/Light mode* with persistence
- *Mobile-first* responsive design
- *Demo login* for all user types
